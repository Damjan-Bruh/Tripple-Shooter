Prodeg Tripple Shooter

 What is Tripple Shooter?
In this game, you are a green man with a gun, that has to defend his teritory. But bombs are comming and if they come behind player, you lose. There are three bombs comming, so be careful. To defend your teritory, you need to shoot bombs. Controls are: w and s for up and down, up arrow and down arrow for faster up and down and space for shooting.
About programs you need
Prodeg Tripple Shooter is a NON executable file, so you will need libraries and programs installed. Firstly, you will need Python 3.12 or higher, because it is a python file(Click link named Python to go to python webpage). Secondly, you will need pygame installed, which you can install by opening cmd and typing 'pip install pygame'(You will need python for this to work). And, finaly, you will need to download Prodeg Tripple Shooter by clicking link named Download. You will also need to download IDLE python. Also, you will need to download file from link named BG Download. You Should be able to run Tripple Shooter.
